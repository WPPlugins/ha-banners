Widget Banners manager.

== Description ==

The plugin helps to manage the banners over the WordPress blog. It is very easy to use.

create and manage banners:

*   create a banner is as easy as creating a post
*   create banners rotations
*   widget banners
*   banners statistics.

Translations

*   English
*   Russian

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload the plugin files to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Settings->ha banners screen to configure the plugin
